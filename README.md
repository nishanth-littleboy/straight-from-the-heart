# Straight-From-The-Heart
A beautiful, interactive web-based gift featuring synced lyrics, a starry background, and a special surprise at the end.

 - Synced Lyrics: Lyrics are perfectly timed to the background music using the HTML5 Audio API's currentTime property to ensure they never drift.

 2. Running the Project:
 - Since modern browsers block "Auto-play" and some local file requests, you must run this using a Local Server.

   Option A: Using VS Code
 - Open your project folder in VS Code.
 - Open the terminal (Ctrl + `).
 - Type python -m http.server 8000 and press Enter.
 - Open your browser and visit: http://localhost:8000.
